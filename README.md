This project is a simple Wordle clone web app :)

### Directories

#### worgle-html
*Pure HTML/CSS/JS project*
###### Run instructions
1. `cd worgle-html`
2. Start `Live Server` VS Code extension (or alt server)

#### worgle-react
*React*
###### Run instructions
1. `cd worgle-react`
2. `npm start`
